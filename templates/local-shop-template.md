# Шаблон «Локальная лавка» (по образцу vmestesryboi.ru)

Многостраничный статичный сайт небольшого локального магазина с заявкой
вместо корзины-оплаты. Стек под workflow Панина: чистый HTML/CSS/JS,
каталог из JSON, деплой GitHub → Cloudflare Pages или reg.ru, правки из iSH.

## 3. Ключевые продуктовые решения

- Заявка вместо корзины с оплатой: цена/вес подтверждаются вручную
- Форма работает на сайте, мессенджеры - запасной канал
- Промокод - мем-код, легко запомнить и отследить канал
- Персона владельца сквозняком: слово основателя, «Написать [имени]»
- Локальность как актив: улица в H1, метро с метрами, ориентир двери
- SEO-контур: гиды «как выбрать/приготовить X» ведут на карточки
- Trust: только Яндекс.Карты, живые фото, происхождение товара

## 4. Техархитектура

repo/
  index.html, catalog.html
  categories/*.html - страницы разделов
  guides/*.html - SEO-гиды
  data/catalog.json - [{id,cat,name,desc,origin,price,unit,img,hit}]
  js/catalog.js, cart.js, form.js
  css/style.css - CSS-переменные
  images/{brand,hero,live,gifts,catalog}/
  functions/api/order.js - приём заявки в Telegram

Приём заявки: Cloudflare Pages Functions (order.js -> Telegram Bot API,
env-токен) или reg.ru (order.php через curl). Паттерн уже отработан на
JadeKey-боте.

Корзина: localStorage-массив, счётчик в шапке, кнопки формируют
предзаполненный текст со списком позиций и промокодом. Оплата при
получении, либо YooKassa-ссылки.

SEO на каждой странице: title/description/OG-теги, canonical, Метрика,
schema.org LocalBusiness.

## 5. Чек-лист запуска нового проекта

1. Бриф: ниша, адрес, метро, часы, телефон, TG/WA, промокод, пороги
2. catalog.json: 8-10 категорий, 30+ позиций на старт
3. Палитра: 1 акцентный цвет, лого, системный шрифт
4. Страницы категорий + 5-10 SEO-гидов
5. Приём заявки + уведомление владельцу в Telegram
6. Метрика, robots.txt, sitemap.xml, favicon, OG-обложка
7. Тест формы с телефона
8. Карточка на Яндекс.Картах, сбор отзывов, вставка виджета
